**▶ NEW FUNCTION – [ ITEM LOCK ]**

![](https://clan.akamai.steamstatic.com/images/38551250/ed342b699ed4d3012a7f9002c5b23b13e2ac46da.png)

- Item Lock Function will be applied to Inventory / Profile / Bank / Closet
- Locked Items (Equipment / Costume): Some Functions (e.g. Upgrade / Delete / Trade / Market / Mail / Seal / Wrap / and so on) will be Restricted.

- How to Lock Items
> Item → Mouse Right-Click → Lock
> Item Lock Function Shortcut Control: 'Ctrl' button + Mouse Right-Click

- How to Unlock
> Locked Item → Mouse Right-Click → Unlock
> Item Unlock Function Shortcut Control: 'Ctrl' button + Mouse Right-Click

- Not Possible to Use Lock Function: Time-limited Costume / Time-limited Equipment / Photo / My Room Furnitures

**▶ NEW FUNCTION – [ SPIRIT EXP TRANSFER ]**
- Enable to Transfer EXP from Spirit item to Another Spirit item.

*<u>****</u>*
Spirit A (with EXP 1500) / Spirit B (with EXP 0) → [ Spirit EXP Transfer ] → Spirit A (with EXP 0) / Spirit B (with EXP 1500)

- Which NPC?: NPC ‘Sedurin’ in Arcadia Realm

****
1. Spirit EXP Transfer

![](https://clan.akamai.steamstatic.com/images/38551250/3236ea1e2db3a7ebe2a0891ec400b944deb3a121.png)

- Depends on the Spirit you registered (To provide EXP to Another Spirit item) → 'Which Spirit is possible to receive EXP' will be decided
※ Spirit can only be registered within the specified list.
- Register Spirit : Can register when there is Minimum (10) EXP.
- Register Spirit : Not Possible to register if  Spirit is Max Level and EXP.
- Spirit : (1) States will be RESET. (2) Remain in the profile / inventory. (Not Deleted)
- More EXP you want to Transfer → the More Material Items you will be required.
- Transfer EXP: Exceeded EXP will be LOST.
