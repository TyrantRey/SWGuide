# [Notice] 2021 July Update Preview

![](https://clan.akamai.steamstatic.com/images/38551250/7adcdb549abfe51a1ebfc5934a039625bb2fb588.png)

Hello, this is an announcement from SoulWorker.
Information about 2021 July Updates taking place can be found below.

**---------------------------------------------◆Preview◆---------------------------------------------------**
**☞ July 1st Update **

**▶ Costume Re-packaging System**
- A new function of repacking costumes that have been already obtained will be newly added.
- The package will be tradable or possessed based on being equipped or not when being unpacked.
 > Assuming that there are costumes possessed:
 The package will be possessed. All costumes become possessed when reopening it.
 > Assuming that there are costumes unequipped:
 The package will be tradable. All costumes become tradable when reopening it.

- All costumes enabled to apply the repackaging system can be sealed infinitely when the new function has been applied to the in-game system.
 > Event costumes, Individual costumes (Example: Wings) will not be repacked.
