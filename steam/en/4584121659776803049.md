**▶ NEW SYSTEM – Active Skill Auto-Learn / Mastery System**
- Active Skills will be automatically learned when players reach a certain level.
* SoulPlus of Active Skill: Enable to 'Learn' / 'Change' / 'Cancel' in the Skill tab. (No Skill Reset item is required.)
* Passive Skill (Such as 'Tumble' / 'Retaliation'): Consume SP to learn. (Same as Season 1)
* SoulPlus: Enable to 'Learn' / 'Change' / 'Cancel' in any Districts. (Except in Mazes and District 6)

- Mastery: Consume SP to upgrade Character Stats. (Battle Mastery / Technical Mastery)
> Press Skill ( k ) button → Can find Mastery tab
> You can freely add or subtract SPs until you press the 'Apply' button of Mastery.
> SP requirement for Mastery / Passive Skill: It will be fixed to 'Consume SP 1' regardless of level.
> Skill Reset / Skill Tree Reset: SPs that used for Mastery will be reset / returned.
> Ctrl + (Mouse) Left Click: Can consume (or reset) SP 5 at once
> Shift + (Mouse) Left Click: Can consume (or reset) SP Max at once
> After Update → The max SP amount increases according to the character level.
> SPs that used before Update → ALL RESET

**▶ NEW SYSTEM – Soul Mate**
- Soul Mate is a companion of SoulWorkers. They can do Auto-Loot (items) and provide Buffs to SoulWorkers!
- [ Y ] button is the shortcut key in the game.
- Soul Mate: Account Basis (Soul Mates will show for all characters on your account.)

**1) How to Obtain**
- Can obtain through 'Soul Mate Box'.
- Gather 'Memory Fragment' → 'Release' to obtain
- Cash Shop: Through Soul Mate 'Transmission' system in Cash Shop
- Not possible to hold same Soul Mates: If you obtain existing Soul Mate again → it is automatically converted to a "Memory Fragment"
- Memory Fragment can be purchased at 'Fragment Shop' (Soul Mate Menu) with 'Colorless Memory Fragment: Pet'.

**2) Memory Fragment: Soul Mate**
- Use 'Memory Fragment' to raise a Soul Mate's Rank.
- Can purchase at 'Fragment Shop' (Soul Mate Menu) / 'Gruton Coin Shop' (Mileage Store - Soul Mate) / 'Cash Shop' (Transmission - Soul Mate)
- If a Soul Mate's Rank reach MAX:
> Not possible to obtain Memory Fragments of Max Ranked Soul Mate.
> Can obtain 'Colorless Memory Fragment: Pet' instead.

**3) Summon / Companion**
- Summon: Can summon a Soul Mate that you choose.
- Companion: (e.g.) Summon 'Haru' Soul Mate and Companion 'Erwin' Soul Mate → 'Erwin' Soul Mate follows you but the buffs of 'Haru' Soul Mate is applied to your character.
- Summon / Dismiss: Possible to do in any Districts.
- In the Maze: If you forget to summon a Soul Mate, you can't summon in the maze. You must summon a Soul Mate BEFORE you enter a maze. ('Dismiss' is not possible, either.)
- Not possible to summon any Soul Mates in ‘District 6’.

**4) RANK of Soul Mates**
- Soul Mate Rank: Total of 20 Ranks (C ~ SSS) / Each Soul Mate has a different starting rank.
- Use 'Memory Fragment' to raise a Soul Mate's Rank.
- Rank Up → Soul Mate's Stats will be increased. (Some Soul Mates increase their Stats per Level.)
- Soul Mate 'Story' & 'Voice': Can be unlocked by Rank Up.
- Rank C / Rank B / Rank A: Enable to rank up to ★3
- Rank S / Rank SS: Enable to rank up to ★5
- Rank SSS: MAX RANK (No higher rank exists.)

**5) Rename Soul Mate**
 - Can rename your Soul Mates by using 'Soul Mate Name Change' item. (Can purchase at Cash Shop)

**6) Level of Soul Mate**
- Soul Mate can obtain EXP as much as FP has consumed AFTER MAZE CLEAR.
- Not possible to get any EXP if leaving without clearing the maze.
- Level Up → Soul Mate's Stats will be increased. (Some Soul Mates increase their Stats per Rank.)

**7) Stats of Soul Mate**
- Stats will be increased by Rank / Level.
- 'Max Stats' Button: Enable to check Rank SSS / Lv.30 Stats of Soul Mate
