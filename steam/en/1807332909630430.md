# [Notice] Character Creation Issue (Resolved)

Hello! This is an announcement from Soulworker.

We have identified an issue affecting character creation on certain servers. Currently, when attempting to use a character name that is already taken, no error message or prompt will appear.

Normally, you would receive a message indicating the name is unavailable, but due to this issue, no such notification is displayed.

If you get stuck on the character creation screen, please assume that your preferred name is already taken. You should be able to proceed by entering a different name that has not yet been used.

Our team is working on fixing this as soon as possible.

Notes:
- Affected servers: EN and TC
- CN servers are not affected

We apologize for the inconvenience and appreciate your patience.
