# [Notice] 2022 September Update Preview

![](https://clan.akamai.steamstatic.com/images/38551250/86ee52367c9f5edb5bbf78095ba0b87e3d93b4e8.png)

**☞ September First Update
**
**▶ NEW SYSTEM – Costume CLOSET/ Costume PRESET**

- Costume Preview / Enable to Store Costumes in the CLOSET.
- Can store various costumes (Cash Shop Purchased Costumes / Costumes obtained through Events / and so on) in the CLOSET.
- Dyed Costumes: Can be stored in Each Color and Enable to check through PREVIEW of the Closet.

- Costume PRESET: Can SAVE / LOAD your favorite costume settings.
- How to Activate: Save Costumes in the PRESET SLOT → Load PRESET in  → Your Costume Preset will be ACTIVATED

**▶ NEW SYSTEM – Character Quick Change**
- Change Character (Quick Version): No need to go back to Select Character! Enable to change to your other character IMMEDIATELY!
- Can find 'Change Character (Quick Version)' via SYSTEM MENU.
- Enable to check 'FP' / 'Maze Entry' Info of the Character that has accessed in the game today.

**▶ NEW EVENT – Fireworks Festival**
![](https://clan.akamai.steamstatic.com/images/38551250/8a618307f0b04fca287c7f3ecd20ee8f42eb1800.png)
- New Event Maze: ‘Fireworks Festival’ will be updated.
- Speak to NPC 'Toru' in Grace City → Can receive the Event Quest: ' Summer Night Fireworks Festival'

**☞ September Second Update**

**▶ NEW RAID – Tragic Fairy **
![](https://clan.akamai.steamstatic.com/images/38551250/254a36cfaa6834d50690bc16fc6f144596815405.jpg)
- New Raid: TRAGIC FAIRY will be updated.
-  Mode: Can challenge as Solo (1 Player)
-  Mode: Can challenge as Solo or Party (1 ~ 4 Players)
- New Weapon / New Gear / New Accessory will be updated.
- New Scenario will be added.
- Daily Mission 'Supercar with No Brakes' Quest Condition will be changed.

**▶ CHANGE – Violent Sun / Broken Savior Equipment (Slightly Easy to Craft and Upgrade)**
- 'Picaresque' equipment: Craft / Upgrade will be slightly easier than before due to the update of New Weapon / Gear (Tragic Fairy Raid).
> Upgrade: 'Picaresque' Weapon / Gear will not be destroyed.
> Random Option Change of 'Picaresque' equipment: Basic Material items (that used for crafting the equipment) will be consumed for Random Option Change (No more ‘Energy Converter’).

- Pale Ashes Extend equipment will be added to Crafting List as Time Limited Item (e.g. Can be used for 28 days and then Auto-Deleted).
> Time Limited Pale Ashes equipment can be crafted by Each Character Every Month.

**☞ About October Update**
- Speech Bubbles Customization System will be updated.
- Tragic Fairy  will be updated.
- New Soul Mate: Miriam / Seres
- Character Skill Balance Patch will be scheduled.
- New Akashic Record will be updated.
