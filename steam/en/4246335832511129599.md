**◆ 2022 March Update Preview**

**☞ March First Update**

**▶ SYSTEM CHANGE - BANK**
- 'BANK' system of NPC  will be changed. (Well… It’s about time.)
> Account Basis & Can share with other characters
- NO MORE Previous Bank System (Each Character Basis / Not shared with other characters)

>  100 mails → Receive Items and Delete Mails → Rest of New E-mails will appear

**▶ MORE GAME SYSTEM CHANGES (DUE TO BANK SYSTEM CHANGE)**
※ Many contents related to the balance of the game will be altered.

- Money will be changed to 'Account Basis'.
 > Money: Zenny / Ether / BP
> No More Each Character Basis → After the update, Account Basis!
 > After the update → Zenny / Ether / BP of Each Character → It will be all combined.

- Item Lists: Change to
 > Accessory Box: Purchase from Gruton Coin Shop.
 > Merchants' Union VIP Membership Card / Union Bank V.I.P Card
 > Progress-related reward packages
 > Steel Grave related reward packages
 > Client Lv. of NPC related reward packages
 > Level Up packages (For Events / For Character Creation)
 > Quest-related items
 > Titles that have earned as quest rewards
 > Support Equipment (e.g. Equipment Name start with 'Supportive' ~ / 'Anti-Destructive' ~)
 > Weapon Name start with Primacy / Twilight / Pale Ashes (Only Weapons. Not for Gears and Accessories.)
 > Pale Ashes Hero Weapon Box
 >  Training Center / Ranking Maze / Corrupted Record
 > Character Level Jumping related items and packages
* Items that are currently set as 'Cannot be stored in the Bank' will remain the same as before.

- Daily Attendance Reward / Stay Connected Reward / Consecutive Attendance Reward
 > Reward Check System: (Previous) Each Character Basis → (After Update) Account Basis
 > Attendance Rewards of March 1st / March 2nd, 2022 will be initialized and accumulated from the beginning.

- Daily Missions
 > The Progress Unit will be changed to Account Basis.

- Shop
 > Some Items: (Previous) Character Basis Purchase Limit → (After Update) Account Basis Purchase Limit
 > Collection Plus Manual [10M] / Corrupted Records' Appendix / Handset Transmitter / Violent Sun Mission Receiver / Parable's A.R Transmitter

- League Raid Rewards
 > The amount of League Raid Coins acquired will be changed between Main Character and Non-main Character.
 > Main Character: 5 / 8 / 10 → 5 / 10 / 20 League Raid Coins
 > Non-main Character: 5 / 8 / 10 → 1 / 3 / 8 League Raid Coins

- Cash Shop
 > Some Items: (Previous) Character Basis Purchase Limit → (After Update) Account Basis Purchase Limit
 > Items that change the system of Purchase Limit: The number of purchase limit will be initialized after the update.
 > Special Upgrade Count Box / Tag Starter PKG/ Newbie Booster PKG I~III /  Lunar Fall /  Violent Sun / Seal Support PKG / Gear Upgrade PKG I~III / New Beginner PKG I~III / Challenger PKG / A.R Transmitter PKG III+ / Akashic Record Starter PKG I~II

- Soul Pass
 > The progress of the Soul Pass will be changed → Share by All Characters
> Soul Pass Reward: Can only receive once per account
 > This change will be applied to the next Soul Pass.

- Maze Entry Number
 > Mazes with Entry Limits: (Previous) Each Character Basis → (After Update) Account Basis
 > 'Violent Sun' Maze: Daily Entry Limit per Each Character [ 3(+1) ] → Daily Entry Limit per Account [ 3(+1) ]
 > 'Daydream' Mazes: Daily Entry Limit per Each Character [ 2(+1) ] → Daily Entry Limit per Account [ 5(+2) ]

 >  will be opened irregularly.

- Events
 > Event Progress / Event Crafting Lists / Event Item Rewards: (Previous) Each Character Basis → (After Update) Account Basis

 > This change will be applied from March First Update.
