# [Notice] 2023 January Update Preview

![](https://clan.akamai.steamstatic.com/images/38551250/86ee52367c9f5edb5bbf78095ba0b87e3d93b4e8.png)

**◆ 2023 January Update Preview**

**☞ January First Update (on January 5th, 2023)**

**1. NEW FUNCTION – [ ITEM LOCK ]**
- Item Lock Function will be applied to Inventory / Profile
> Enable to Lock Equipment / Costume
> Locked Items (Equipment / Costume): Some Functions (e.g. Upgrade / Delete / Trade / and so on) will be Restricted.
> Item Lock Function Shortcut Control: 'Ctrl' button + Mouse Right-Click
> Locked Items can be UNLOCKED by the 'Unlock' function.

**2. NEW FUNCTION – [ SPIRIT EXP TRANSFER ]**
- Enable to Transfer EXP from Spirit item to Another Spirit item.

**<u>**</u>**
Spirit A (with EXP 1500) / Spirit B (with EXP 0) → [ Spirit EXP Transfer ] → Spirit A (with EXP 0) / Spirit B (with EXP 1500)

- Depends on the Spirit you registered (To provide EXP to Another Spirit item) → 'Which Spirit is possible to receive EXP' will be decided

- Spirit EXP Transfer: No Limit to the number of times. However, some EXP may have a chance to be lost.
- More EXP you want to Transfer → the More Material Items you will be required.
- EXP transferred Spirit item will be remained (in the profile / inventory) without being deleted.

**3. NEW COSTUME – [ FOREIGNER QUARTET ]**
- New Costume  will be updated.
> Costume Theme: Second Stair's Infamous Vessi
> 'Special Gesture' will also be included!

![](https://clan.akamai.steamstatic.com/images/38551250/def132c0aff7609cad79b321b4e7af3c131359b4.png)

**☞ January Second Update (on January 19th, 2023)
**
**1. NEW SYSTEM – AKASHIC RECORD REGISTER **
- Akashic Record Register means:
> Akashic Record (Inventory) → Click to USE → that Akashic Record goes to (Akashic) Menu
> *<u>**(After January 19th Update)**</u>* All your characters (Account) will share the state of Akashic Record Register.

*<u>****</u>* 'Vista' Akashic Record (Inventory) → Click to USE → 'Vista' Akashic Record goes to (Akashic) Menu
★ All your characters (Account) will be able to use 'Vista' Akashic Record on (Akashic) Menu.

- HOWEVER, Akashic Record Equip → It is saved for Each Character in the same way as before.

- Akashic Record Remove means:
> Akashic Record (Akashic Menu) → Click to REMOVE → that Akashic Record returns to (Inventory)
> *<u>**(After January 19th Update)**</u>* All your characters (Account) will share the state of Akashic Record Remove.

*<u>****</u>* 'Vista' Akashic Record (Akashic Menu) → Click to REMOVE → 'Vista' Akashic Record returns to (Inventory)
★ All your characters (Account) can't find 'Vista' Akashic Record on (Akashic) Menu. (Enable to find in the Inventory)

**2. NEW GRADE (AKASHIC RECORD) – [ MYSTIC ] GRADE**
- New Grade for Akashic Record [★5] will be updated.
> Name of New Grade: Mystic

- FUSE / ENGRAVE (NPC 'Eo') → there will a new fusion tab for Mystic grade.
> (Current) EXP Fuse / Merge / Engrave / Hidden Fusion
> (After January 19th Update) EXP Fuse / Merge / Engrave / Hidden Fusion / 'Mystic Fusion'

-  Menu: Only some of Akashic Record [★5] (Hidden / +2 state) are available for this menu.
- Only One of Akashic Records will be announced as the First Mystic Grade A.R Card.

**3. NEW FUNCTION – [ DAMAGE REPORT ]**
- Damage Report: Allow you to check your Damage Contribution

- Enable to check your Damage Report on the Result Screen (Raid).
> Damage (Contribution) Report Ranking No.1 Player will become [ MVP ].
> *<u>****</u>* Solo Raid Mazes → No MVP

- About Damage (Contribution) screen
> What Skills you have used
> Number of Times each skill has been used
> Total Damage per Each Skill
