# [Notice] 210513 Update Note Preview

![](https://clan.akamai.steamstatic.com/images/38551250/3adcd2dc241ff4bcc2c84ee33eb6396c0aa19121.png)

Hello. This is from SoulWorker.
We are here to inform you about the May 13th updates and event contents.

---------------------------------------------◆ Update Contents ◆-----------------------------------------
**1. Opening new server and adding ‘English’**
- New language, ‘English’ will be added.
 > Right click Steam Client’s SoulWorker – Properties – Enable to select ‘English’, or ‘Traditional Chinese’ at Language tab.
 > Please be notified that some fonts in few countries may not be printed properly based on the selected language.
- New Server ‘ Tenebris’ will be added.
 > World Boss (The Doom, The Thing) don’t appear in ‘ Tenebris’ server; we will provide further information about World Boss Update in the near future.
 > The previous server name will be changed from ‘Rosca’ to  Rosca.

**2. New Functions in Worker Diary**
![](https://clan.akamai.steamstatic.com/images/38551250/43ab49b610e70a1c4034bab12475d0e77e502857.png)
- New function for viewing the gameplay records of Account/Character.
- Enable to confirm by clicking the new icon at the top right corner of the character information screen.
- Regarding gameplay information other than Account/Character game records, it will be applied with game records that have been created after May 13th update contents.

**3. New Channel Select Helper Function (*  Tenebris Server Only)**
![](https://clan.akamai.steamstatic.com/images/38551250/f78f876b93117926388820b805cd538a246054ec.png)
- When entering the channel, you will have automatic settings which allow you to enter the channel that is the easiest to enter, based on the region you selected.
- Following regions has been refered below.
 > U.S. East(USE) / U.S. West(USW) / Germany(GWC) / UK(UKS) / India(IDN) / Brazil(BRA) / Australia(AUS)

**4. Modification on SoulCash Top-up System**
![](https://clan.akamai.steamstatic.com/images/38551250/de897bcd970af50106f86d76a2f0379f1b7ef3fa.png)
- Key currency will be changed from TWD to USD, as well as the payment charge unit.

**5. Other Adjustments**
![](https://clan.akamai.steamstatic.com/images/38551250/efed37028040ae226ea93d8964b8b53e7be16ad5.png)
- Login title screen has been changed.
- SS Rank has been changed to S+ in the game.
 > Contents will be changed according to the following changes and updates.
 > Costume ‘SS Icon’, ‘Premium SS Icon’ design and explanation have been changed to ‘S+’.
 > SS in Achievement explanation has been changed to S+.
 > SS in Maze Clear Rank Image has been changed to S+.
- The problem with incorrect option content output in the settings has been modified

---------------------------------------------◆ Event Content ◆--------------------------------------------

**1. ‘Global Launching Celebration Cake’ event will start soon.** (From May 13th Update ~ June 10th Update)
- Enable to obtain ‘ Celebration Cake’ based on the consumed FP, after completing the maze. (Maximum daily acquisition: 400 per character.)
- Enable to exchange the obtained item ‘ Celebration Cake’, to various items, including ‘Fairy Wing Costume Select Option’.

**2. ‘Global Launching Goods Ticket’ event is starting.** (From May 13th Update ~ June 10th Update)
- Enable to obtain ‘ Goods Ticket’ with various methods.
 > Daily Mission: Log-In
 > Event Daily Mission: Maze Clear 3 times.
 > Event Daily Mission: Raid Clear 3times (Casual Raid , Golden Citadel , The Primal , Alter of Void , Hidden Hideout, Lunar Fall)
 > Stay-Connected Reward 60 minutes.
- For further details, please refer to the additional upcoming announcement.

 > Level 60 : Ether 300,000, Energy Converter x 15, Anti-Destruction Device x 20, Primacy Soul Weapon +9 Package
 > Level 65 : ★5 Akashic Record Select Option x 1

2) Costume Try-On Count Box
 -  Count Box which can be used once every 24 hours will be given, different items appear for each opening.
