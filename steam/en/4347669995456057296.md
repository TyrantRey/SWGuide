3) Victory Pose
- This new system allows setting Victory Pose for each character.

![](https://clan.akamai.steamstatic.com/images/38551250/43e5ce7b8f0d367622bb5c6485b611ca6d56447b.png)

- Press Skill tab (K button) → Can find Victory Pose and configure a character's Victory Pose.
- Victory Pose Tab: Can check all the Victory Pose Lists. (Victory Pose that you don't have → Locked)
- Can equip more than one Victory Poses.
- If equip more than one Victory Poses → One of the equipped Victory Poses will be randomly displayed when clear the maze.
- After the update → Default setting of Victory Pose: All Random
- If equip any Victory Pose → All Random will be disabled.
- No Victory Pose is equipped → All Random will be activated again.

 ※ Can choose 2 basic victory poses for each character.
※ NPC : Can craft additional Victory Pose.
