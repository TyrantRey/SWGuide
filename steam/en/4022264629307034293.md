# [Notice] 210819 Update Note Preview

![](https://clan.akamai.steamstatic.com/images/38551250/7adcdb549abfe51a1ebfc5934a039625bb2fb588.png)

Hello, this is an announcement from SoulWorker.
Information about 2021 August 19th Updates taking place can be found below.

-------------------------------------------◆ Update ◆---------------------------------------------

**1. “God’s Blessing on this Wonderful World!” Collaboration**
- Collaboration with “God’s Blessing on this Wonderful World!” will soon be starting.
- 4 Event Quests ‘Vagabond of Wonderful World I’, ‘Vagabond of Wonderful World II’, ‘Vagabond of Wonderful World III’, ‘Path back to Wonderful World’ will be added. (Limited to only Lv. 40 and above)
- Event Quest ‘Vagabond of Wonderful World III’ can be received from Toru after completing ‘Vagabond of Wonderful World II’
- In-game Event ‘Vagabond of Wonderful World’ will be available.
- Costumes and Photos only available during ‘God’s Blessing on this Wonderful World’ will be added.

**2. Fixes**

- Changes to the Login Screen will be made.
- Chatting/Name filters will be updated.
- Issues regarding Social Item UI when closing and opening Cash Shop while using Furniture Items have been fixed.
- Issues regarding Increased Damage Buffs when using Stella’s ‘Decayed Pulse’ have been fixed.
- Issues regarding inconsistent SoulStrike Attacks when using Stella’s ‘Beads Play’  have been fixed.
- Issues regarding Character Lee Nabi not being able to use Items in ‘  Nostalgic Kiddy Uniform’ have been fixed.
- Issues regarding progression of Mazes being intermittently reset have been fixed.
- Issues regarding Gatling Gun positioning when using Nabi’s ‘Chainsaw’ have been fixed.
- Issues regarding an abnormal Portal visible in Dipluce Horizon have been fixed.
- Issues regarding Nabi’s Evasion Voice when using ‘Desire Worker’ Equalizer have been changed.
- Issues regarding part of the dialogue not appearing during Quest ‘Marching of Blue Light VI’ when in ‘English’ Language settings have been fixed.
