# [Notice] [Kent] CN Server Input Method Issues

![](https://clan.akamai.steamstatic.com/images/38551250/d0f343240d222afcf346c478d79025ad8d836f7d.jpg)

Hello! This is an announcement from Soulworker.

Previously, we received feedback from players regarding the lack of a selection box when using input methods in the game on the CN server. Today, we have optimized the input method, and players can now communicate and use Microsoft Pinyin in the game. We will continue to work on optimizing the Sogou and QQ input methods in the future.

Please note that the input method fix requires you to log back into the game for the changes to take effect. We recommend logging in again to apply the update.

Thank you for your understanding and support.
