# [Notice] Additional explanations for New User Package Issue

![](https://clan.akamai.steamstatic.com/images/38551250/3adcd2dc241ff4bcc2c84ee33eb6396c0aa19121.png)

Hello, this is an announcement from SoulWorker.

We have checked that some packages have not been clearly given out and addressed these issues in our official Steam Announcement.
We internally investigated New User Package issues since some packages are not appropriately distributed. We have addressed these issues in our official Steam Announcement.
However, we still got reports from users receiving the New User Package without costume box to request a re-gift.
Those 3 New User Packages are given to New Players as special gratitude of Global Service Expansion.
Please be notified that current SoulWorker users are not qualified for these packages.
All Packages will be separately given to Rosca & Tenebris' new users joined (May 13th~ 27th) only and delivered altogether after the regular maintenance.

****

- Any SoulWorker users who have not received New User Packages
※ Unable to receive New User Packages
※ Any related request for a Refund is not acceptable for this case.
※ Rewards of New User Packages get differentiated based on the Date you create the first character; Additional measures are not applied.

(Example)
1. Create a SoulWorker account after November 2020
   -> Being qualified for the New User Package I
2. Create a SoulWorker account from May 13th ~ to June 10th, 2021
   -> Being qualified for All 3 New User Packages
3. Create a SoulWorker account after June 10th, 2021
   -> Being qualified for the New User Package I

** **
All packages will be delivered all at once after the maintenance.
We always try to return your love and supports by providing the best service we capable of.
Thank you.
