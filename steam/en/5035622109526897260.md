# [Notice] 2023 February Update Preview

![](https://clan.akamai.steamstatic.com/images/38551250/86ee52367c9f5edb5bbf78095ba0b87e3d93b4e8.png)

**◆ 2023 February Update Preview**

**☞ February First Update (on February 2nd)**

**1. NEW FUNCTION – [ League Booster ]**
- League Booster will be added.
> Purchase the League Booster → League Booster will be activated for a period of time. → League members can use this League Booster
> (Purchase - Default) It can be purchased by League Master.
> However, League Master can provide a purchase permission to other members (such as Vice / Elder).

**2. CHARACTER BALANCE PATCH**
- Balance Patch is scheduled for Some Characters.
> Skill Balance
> Some Improvements

- Damage Up / Super Armor Break Up for Some Skills
- Improve the convenience of Some Skills
- Counter Upgrade Applied Skills → Damage Up
- 'Heavy Drop' Skill: Charging Time - Shorten / Charge Phase - Reduced

- 'Limit Break' Skill: Buff Effect will be changed.

- SoulStrike Mastery: Mark Explosion / Recover HP will activate faster than before.
- Damage Up for Some Skills

- Duration of 'Limit Break' Buff will be reduced. But, Damage Up / Super Armor Break Up.
- 'Titan's Massacre' Skill: Skill Delay Time will be improved.

- Turret (SoulStrike of 'Honey Bee'): Turret's Attack Speed Up (Duration Only)
- ‘Last Strike’ / ‘DestroNail’ Skill: Skill Delay Time will be improved.

- Some Skills' Damage / Super Armor Break value will be changed.
- Some Skills' Cooldown Time will be changed.
- Improve the convenience of Some Skills
- 'Fatal Slash' / 'Dark Breaker' Skill: Sword Aura's firing distance will be reduced.

**3. IMPROVEMENT - BROOCH SET BONUS**
- If Equipped Brooches are All Different Grade / But SAME THEME → Enable to get the Set Bonus of the Lowest Grade's Brooch

***<u></u>***
> Equipped: MEGA 1 / GIGA 1 / TERA 1 + ALL SAME THEME → Get Set Bonus of MEGA Brooch
> Equipped: MEGA 1 / TERA 2 + ALL SAME THEME → Get Set Bonus of MEGA Brooch

![](https://clan.akamai.steamstatic.com/images/38551250/0d168c1ccbd76169914715750a913d34d820a615.png)

**4. IN-GAME IMPROVEMENTS**
- H.E.M.S: Enable to Select Mails that you want to get attachments

- Ranking UI: Ranking Maze Entry Button will be added.
> Enable to enter if you have a ' Challenge Ticket' that requires each Rank Mazes.

- Inventory → Select Option Boxes that you want to open (Select Item / Decide Number of Boxes) → Enable to open at once

*<u>** **</u>*
 Soul Mate Select Option Box x100 → Choose Soul Mate 'Dhana' → Decide How Many Boxes you want to open (Select Option Box x30) → Click 'Receive' button → You  can get Soul Mate 'Dhana' x30 at once.

**☞ February Second Update (on February 16th)**

**1. Ruin Fortress  Mode**
- Ruin Fortress  Mode will be added.
> Level Requirement: Lv. 72 or Higher
- Ruin Fortress  Mode can be entered through the portal of Forgotten Armory / Deep Core / Acheron / Ark Ship.
- Material Items that gained from HERO mode: Enable to get  Costume and Rewards (Gruton Coin Shop)
>  Costume is available for All Characters (Including Dhana Opini)

![](https://clan.akamai.steamstatic.com/images/38551250/158260dc65be5e76ca31e18234ac872cd9df4858.png)

**2. BIRTHDAY COSTUME - Lily's Beautiful Holiday**
- February 27 is Lily’s Birthday.
- Lily’s Birthday Costume  will be added.
