# [Notice] 210708 Update Note Preview (July 9th Edited)

**--------------------------------------------◆ Updates ◆-------------------------------------------------**

**1. Costume Re-packaging System (Costume Wrapping) **
- A new function of repacking individual costume parts stored in your Fashion inventory will be newly added.
- Costumes can be re-packed as a package to use ‘Costume Wrapping’ when all Costume parts are owned.

![](https://clan.akamai.steamstatic.com/images/38551250/6fea537fc00b434caca88e428d7598fb84da0cf6.png)
![](https://clan.akamai.steamstatic.com/images/38551250/17c50411c7c1c2a2045c54951c0393026643266e.png)

- Re-packaging will only be available for SoulCash-purchased costumes.
- Costumes with brooches can be re-packed when the brooches get removed.
- Dyed Costumes can be re-packed again, remained as dyed color.
- The package will be tradable or possessed when being re-packed.
> **Assuming that there are any costume parts possessed:** The package will be possessed and untradeable; all costumes become possessed when reopening it.
> **Assuming that there is any costume parts unequipped:** The package will be tradable, and all costumes become tradable when reopening it.

![](https://clan.akamai.steamstatic.com/images/38551250/263624c4f24c8ad7626fa02c116f1ab30839d02f.png)

- It will be easier to recognize Unpacked/Re-packed Costumes mentioned above:
- Costume packages will be purchasable in  –  Category when being registered to the Market.
☞ Our Operational Policy for costumes will be modified with the new function of the re-packaging system. (Costume Wrapping)
- SoulCash-purchased costumes will not be possessed until being unpacked.
- SoulCash-purchased costume packages can be tradeable itself after purchasing them.
- Re-packable Costumes can be re-packed indefinitely.
> Please be notified that some costumes cannot be re-packed when included in the package; Limitations will exist for those costumes.
- SoulCash-purchased costumes cannot be refunded when being unpacked or traded in the Market.
