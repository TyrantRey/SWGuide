# [Notice] 2021 December Update Preview (11/30 Edited)

![](https://clan.akamai.steamstatic.com/images/38551250/6175be722a1c34cc5772f361753890c817e9e5c9.png)

**◆ 2021 December Update Preview**

**☞ December First Update**

**▶ NEW PRODUCT - SOULWORKER PLUS**
- Time Limited Product: Can obtain additional bonus while playing SoulWorker.
- It contains 'Maze Entry Number Increase', 'Additional Stay Connected Rewards', 'Free Respawn Number Increase', and more. The detailed information will be announced via December Update Note.

**▶ NEW MAZE – ‘RUIN FORTRESS HERO’**
- Ruin Fortress Hero will be newly added. Bosses of Ruin Fortress have become stronger here. Defeat them and get special materials!
- Before entering the maze, there will be challenge options such as strengthen the monster or set specific conditions. Clear the maze with challenge options to obtain more and more special materials.
- Above Lv. 72 can enter. New event costumes can be crafted with the special material.
- Ruin Fortress Hero will be open for a limited time only.

**▶ NEW AKASHIC RECORD**

![](https://clan.akamai.steamstatic.com/images/38551250/7df94d40425db5c6f94ef32e51094212a9d69641.png)
- **New Akashic Record: ★5 ‘Chloe’ and ★4 ‘Nabi’**
- After December First Update, ★5 ‘Chloe’ / ★4 ‘Nabi’ will be added to A.R Card Special Select Option and Parable's A.R Transmitter which can be purchased at Mileage Shop. Please note that new akashic records will not be added to A.R Transmitter (Mileage Shop Product) / Select Option (Mileage Shop Product) that was purchased before the update.
-  Low, Normal, High-Grade, Hidden Parable's A.R Transmitter (Not Mileage Shop Product) that you already own: ★5 ‘Chloe’ and ★4 ‘Nabi’ have chances to obtain after the update.

▶** SKYCLOCK PALACE - CARD KEY REWARD CHANGE**
- Among the Card Key rewards of Skyclock Palace, 'Gruton Coin' will be changed to 'Bluelight Dust'.

**☞ December Second Update**

**▶ CHARACTER SKILL CHANGE: IRIS YUMA**
- 2 skills of Iris will be changed to new skills.
1) Spiral Pierce → Fortress Cannon
2) Full Burst → Liberator
The detailed information of new skills will be announced via December Update Note.
