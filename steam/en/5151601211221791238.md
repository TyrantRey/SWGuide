# [Notice] 2023 June Update Preview

![](https://clan.akamai.steamstatic.com/images/38551250/86ee52367c9f5edb5bbf78095ba0b87e3d93b4e8.png)

**◆ 2023 June Update Preview**

**☞ June First Update (on June 8th)**

**★ UPDATE SCHEDULE**
- June First Update was scheduled to run on June 1st, 2023.
- However, it has been changed to June 8th due to internal reasons.

**★ MAY UPDATE EVENT**
- Some of May Events will be closed on June 1st, 2023.
- However, May Events can be found in the game before the update on June 8th.
> Not Possible to obtain Event Items.
> Not Possible to use Event Crafting.

- May Events will be Removed from the game after June 8th Update.

**1. IMPROVEMENTS**
- Upgrade / Refine System
> Current: Need to UNEQUIP weapon / gear to use Upgrade / Refine
> After Update: NO NEED to unequip weapon / gear to use Upgrade / Refine

- Upgrade: Less Money / Asset Required
> For: Some of Equipment (Not All Equipment)

- Tag Options
> Tags with Specific Targets (e.g. Golden Menace / Primal Powers / Parallel Keeper) → Those Tags will be removed from Random Tag Option Items / Select Options.

- Less FP Consumed
> For: Season 1 Area related Mazes

**2. NEW COSTUME**
- **[ Lord Fairytale / Lady Fairytale ]** costume will be updated.
