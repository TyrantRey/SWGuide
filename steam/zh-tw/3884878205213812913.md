# 【公告】商品說明錯誤

![](https://clan.akamai.steamstatic.com/images/38551250/92370b6d3cc78c02e9b321d6306fe6d324a77917.png)

大家好，我們是LION GAMES。
商城中販售之付費道具有錯誤說明，詳細內容如下：

〔錯誤內容〕

**-攻擊型標籤改造晶片禮包 x 30**
![](https://clan.akamai.steamstatic.com/images/38551250/94ca3c948ff8e14860328e51ac2ad454d22da0f4.jpg)
錯誤說明：
1~6階 攻擊型標籤傳送器 x 10
高級攻擊型標籤傳送器 II x 3 (獎勵)

正確說明：
1~6階 攻擊型標籤傳送器 x 30
高級攻擊型標籤傳送器 II x 3 (獎勵)

**-防禦型標籤改造晶片禮包 x 30**
![](https://clan.akamai.steamstatic.com/images/38551250/d47f4fbd039791e8babb52c9b8e1c54a4348cadd.jpg)
錯誤說明：
1~6階 防禦型標籤傳送器 x 30
高級防禦型標籤傳送器 II x 1 (獎勵)

正確說明：
1~6階 防禦型標籤傳送器 x 30
高級防禦型標籤傳送器 II x 3 (獎勵)

購買商品後，禮包內道具數量及運作雖正常，但錯誤的敘述可能造成玩家誤解，敬請留意。
此錯誤會盡快進行修正處理。

造成各位的不便非常抱歉。

謝謝大家。
