# 【公告】艾芙妮爾相關活動公告

![](https://clan.akamai.steamstatic.com/images/38551250/ef6150adfea74e09ee2bc708cdef4ff485fd2d6b.png)

各位新玩家老玩家們大家好，我是GM琪。
為了歡迎艾芙妮爾，除了每日登入獎勵、連續登入獎勵和上線時間達成獎勵外，我們還準備了豐富的禮物喔！
只要完成指定行動，即可獲得以下獎勵。

**◆每日任務（11/12更新後~11/26更新前）**

<u>※每日任務前往過“格雷斯城”一次以上之的角色才會顯示</u>

STEP 1：
於遊戲內畫面右上角按下![](https://clan.akamai.steamstatic.com/images/38551250/0fb57c2100cd428bd5033f67f68e21c588771e53.png)即可於![](https://clan.akamai.steamstatic.com/images/38551250/3465dddc835a6a3514287e2ea2eb90a81cecc796.png)選單中確認「艾芙妮爾上線紀念每日任務」之內容。
按下「接受」接收並完成任務後，即可獲得【EV】輪盤幣，並於角色信箱(收件箱)中領取。

STEP 2：
於遊戲內畫面右下角按下「輪盤」，即可憑【EV】輪盤幣可參加抽獎活動，每次消耗1【EV】輪盤幣
抽中的獎勵將會透過角色信箱(收件箱)中發放。

STEP 3：
若獲得艾芙妮爾券，可於浣熊商店的製作功能製作道具。
可製作道具清單請洽浣熊商店～
![](https://clan.akamai.steamstatic.com/images/38551250/874c380faa6a6e8b7559dd9225e2010064c478d0.png)

**◆艾芙妮爾升級禮包活動（11/12更新後~12/10更新前）**

創建艾芙妮爾角色時，即可獲得「艾芙妮爾新手玩家物品寶箱」。
角色升級時，即可打開背包內相對應等級的獎勵禮包。
除了原有的升級獎勵外，這次特別準備了獎勵禮包喔！

【EV】艾芙妮爾20級獎勵禮包 : 乙太 100,000 / 乙太碎片x100 / 附加獎勵鑰匙x20
【EV】艾芙妮爾30級獎勵禮包 : 乙太 100,000 / 乙太碎片x100 / 防止強化破壞裝置x5
【EV】艾芙妮爾40級獎勵禮包 : 乙太 200,000 / 初級乙太x60 / 防止強化破壞裝置x10
【EV】艾芙妮爾50級獎勵禮包 : 乙太 200,000 / 初級乙太x60 / 防止強化破壞裝置x15 / 全體技能初始化券x1
【EV】艾芙妮爾55級獎勵禮包 : 乙太 300,000 / 新歷史的A.R卡選擇券【5星】 I x1 / 能量轉換器x15 / 防止強化破壞裝置x20 / 卓越審判靈魂武器+9禮包x1

**◆艾芙妮爾時裝體驗活動（11/12更新後~12/10更新前）**

創建艾芙妮爾角色時，即可於背包(靈魂幣)內獲得【EV】時裝限時體驗寶箱![](https://clan.akamai.steamstatic.com/images/38551250/1fe1dce468f2d77ed8342ef3616b53b04ed4332a.png)。
開封後，將可根據打開次數體驗限時時裝，並選擇一套永久時裝。

* 第一天：【EV】永久時裝選擇券![](https://clan.akamai.steamstatic.com/images/38551250/103d93ad0ef9794b848b964ba93abde492a6d9e4.png)， 勳章插槽增加券
* 第二天：童心洋溢者校服 1天
* 第三天：春日運動會 1天
* 第四天：魅魔 1天
* 第五天：暗影輕語 1天
* 第六天：尼希王國騎士 1天
* 第七天：菁英貴族學院校服1天
* 第八天：冬之風 1天
* 第九天：白沙海岸 1天

**◆遊戲內加成活動（11/12更新後~11/26更新前）**

艾芙妮爾在戰鬥的過程中，經驗值、道具掉落率、金幣獲取值將相對有所提升。

那麼，現在就馬上進入遊戲與艾芙妮爾會合吧！
