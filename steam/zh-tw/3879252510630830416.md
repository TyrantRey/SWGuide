# 【公告】琪的過冬準備獎勵活動

![](https://clan.akamai.steamstatic.com/images/38551250/ef6150adfea74e09ee2bc708cdef4ff485fd2d6b.png)

不知不覺就到了十二月，馬上就是冬至了～
雖然琪不用冬眠，但冬天會不自覺地想要囤貨呢
為了讓大家的背包也裡有滿滿的禮物，接下來的時間也和琪一起在遊戲內相見吧！

接下來的一周內，只要登入遊戲，就可以拿到禮物喔
12/10(四)：鳳梨口味糖果50個 + 復活裝置5個
12/11(五)：野餐墊1個 + 【許可證】原始勢力1個
12/12(六)：乙太50000寶箱2個 + 能量轉換器1個
12/13(日)：乙太50000寶箱4個 + 能量轉換器2個
12/14(一)：附加獎勵鑰匙10個 + 新歷史的A.R卡傳送器5個
12/15(二)：附加獎勵鑰匙20個 + 新歷史的A.R卡傳送器10個
12/16(三)：鳳梨口味糖果100個 + 復活裝置10個
12/17(四)：野餐墊1個 + 幸運日 【30M】1個(物品掉落率50%增加道具)

除此之外，平日晚上7~11點還有金幣掉落率增加(+50%)的活動喔！

※注意事項
-每日的禮物將於晚上七點發放，最後領取時間為隔天早上9:00，須於此時段登入方可得到獎勵(12/11新增說明)
-「【許可證】原始勢力」道具為入場次數重置道具，須達到每日進場次數之上限方可使用
-每日獎勵內容現單一帳號領取，請留意領收之角色
