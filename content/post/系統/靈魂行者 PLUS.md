> [!note]
> 靈魂行者 PLUS 是**月費訂閱商品**，在期限內提供多項養成加成與副本福利。**購入後不得退換**。

![](plus.png)

## 加成效果

| 加成         | 效果                  |

| 結算獎勵經驗值    | +20%                |
| 金幣獲得量      | +20%                |
| 結算獎勵金幣     | +20%                |
| 乙太掉落率      | +20%                |
| 道具掉落率      | +20%                |
| 強化成功率      | +50%                |
| 交易所銷售手續費   | 折 4%（原 9 折 → 5 折）   |
| 通關情況點數     | +100%               |
| 城鎮移動速度     | +50%                |

## 副本福利

- **PLUS 專用行動力 +200**（角色分別計算）。
- **每日免費復活次數 +2**（5次）。
- 副本通關時**自動獲得第一階附加獎勵**，無需消耗附加獎勵鑰匙。

## 在線獎勵

- 依**日／週／月**累積連線時間發放獎勵。
- 日累積：**30 / 60 / 90 分**各發送一個 PLUS 獎勵箱。
- 期限內單一帳號可獲得 1 次，達成各階段時自動發送。
