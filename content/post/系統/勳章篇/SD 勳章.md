## 型號簡介

**SD（標準型）** 是最早接觸到的勳章型號，定位泛用、取得門檻低，常作為前期過渡。湊套裝時多用在**手**、**腳**部位。

## 常見攻擊選項

> [!warning]
> 以下選項名稱與數值為社群常見整理，會隨版本與翻譯變動，實際以遊戲內為準。

| 選項 | 效果取向 |
| ---- | -------- |
| 滲透 | 提升貫穿相關 |
| 獨輪 | 約 +5% 對 BOSS 傷害 |
| 突擊 | 攻擊力相關 |
| 熔解 | 攻擊力＋爆擊傷害 |

## 選擇建議

> [!note]
> 同型號的攻擊選項，挑選原則不是「哪個數值最高就無腦堆」，而是**補足角色當前不夠的屬性**。例如貫穿已經夠高，就改補對 BOSS 傷害或爆擊，整體收益更好。

- **獨輪** 的對 BOSS 傷害在 B 傷不足時很實用。
- **熔解** 同時給攻擊與爆傷，較全面。
- 實際取捨需對照自己現有的 B 傷／貫穿數值評估。
