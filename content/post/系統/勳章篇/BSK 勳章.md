## 型號簡介

**BSK（狂戰型）** 的特色是**攻擊與防禦選項都很齊全**，尤其防禦端的「受擊觸發」與「迴避觸發」選項是其招牌。常見配置在**手**、**褲**、**腳**等部位。

## 常見選項

> [!warning]
> 以下選項名稱與數值為社群常見整理（部分來自簡中攻略），會隨版本與翻譯變動，實際以遊戲內為準。

### 攻擊型

| 選項 | 參考效果 |
| ---- | -------- |
| 灰熊 | 約 +48% 攻擊力（約 +800 攻擊），另約 +640 爆擊傷害 |
| 熔解 | 攻擊力＋爆擊傷害 |

### 防禦型

| 選項 | 參考效果 |
| ---- | -------- |
| 狂暴 | 受擊觸發類（約 30%） |
| 欺騙 | 受擊觸發類（約 30%） |
| 激怒 | 迴避觸發類（約 30%） |
| 犀牛 | 迴避觸發類（約 30%） |

## 選擇建議

> [!note]
> 攻擊端 **灰熊**的攻擊與爆傷加成相當可觀，是 BSK 常見主力。防禦端「狂暴／欺騙」走受擊流、「激怒／犀牛」走迴避流，依角色生存需求二選一即可，數值相近時挑套裝湊得齊的那個。
