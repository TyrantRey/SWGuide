## 型號簡介

**SIN（刺客型）** 以攻擊、**攻速**與**爆擊**取向為主，常見配置在**褲**部位（與 BSK 互為選擇）。

## 常見選項

> [!warning]
> 以下選項名稱與數值為社群常見整理（部分來自簡中攻略），會隨版本與翻譯變動，實際以遊戲內為準。

### 攻擊型

| 選項 | 參考效果 |
| ---- | -------- |
| 魯莽 | 攻擊力相關 |
| 猝死／暗殺／刺激者 | 約 +6% 爆擊機率 |
| 根除／清除 | 約 +980 爆擊傷害 |

### 功能型

| 選項 | 參考效果 |
| ---- | -------- |
| 颱風 | 攻速約 +4% |
| 擦傷 | 攻速約 +4% |

## 選擇建議

> [!note]
> 爆率不足時優先「猝死／暗殺／刺激者」補爆率；爆率夠了改挑爆傷（根除／清除）。攻速選項「颱風／擦傷」數值相近，挑套裝湊得齊的即可。
