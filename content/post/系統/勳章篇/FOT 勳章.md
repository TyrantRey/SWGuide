## 型號簡介

**FOT（要塞型）** 以攻擊與**爆擊相關**選項見長，是不少流派的核心輸出來源。常見配置在**頭**、**胸**部位。

## 常見攻擊選項

> [!warning]
> 以下選項名稱與數值為社群常見整理，會隨版本與翻譯變動，實際以遊戲內為準。

| 選項 | 效果取向 |
| ---- | -------- |
| 燃料庫／投石器 | 攻擊力相關 |
| 支配 | 約 +5% 爆擊機率（換算約 +1.67% 傷害） |
| 占領／加農 | 攻擊／爆擊相關 |
| 閃電戰／發射器 | 攻擊／爆擊相關 |

## 選擇建議

> [!note]
> **支配**在爆擊機率尚未滿的角色身上收益明顯（補爆率比硬堆攻擊更划算）；若爆率已接近上限，則改挑其他攻擊選項。一樣遵守「補不足、湊套裝」的原則。
