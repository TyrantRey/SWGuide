## 勳章是什麼

勳章是鑲嵌在**防具**上的小元件，提供攻擊、防禦或功能性的數值加成。每件防具有 3 個勳章孔，全身湊起來就是一筆可觀的屬性，是角色最主要的加點來源之一。分為可交易及不可交易。一般勳章分為~~BIT,KB,MB~~,GB,TB 及角色技能勳章PB

> [!important]
> - 勳章的精髓在「**套裝效果**」：同一件防具上的 3 個勳章孔，若鑲嵌**相同型號且相同稀有度**的勳章，就會額外觸發套裝加成。所以配勳章不是看單張數值最高，而是看整套湊不湊得齊。

## 四種型號

勳章依外型分成四種型號，彼此的套裝效果不同，**只能同型號互相比較**，不能跨型號混搭觸發套裝：

| 型號 |
| ---- |
| SD   |
| BSK  |
| FOT  |
| SIN  |

## 稀有度（由弱到強）

勳章稀有度借用了資料量單位來命名，顏色由弱到強為：

| 稀有度 | 顏色     |
| ------ | -------- |
| BIT    | 綠       |
| KIRO   | 藍       |
| MEGA   | 黃       |
| GIGA   | 紅       |
| TERA   | 紫       |
| Peta   | 角色限制 |

> [!tip]
> 一般從 **MEGA（黃）** 開始往上挑就好，MEGA 以下的數值增幅太低，主要拿來當合成材料。套裝效果也吃稀有度，所以盡量讓整件防具的 3 顆同色。

## 屬性分類

每種型號的勳章又分成三類，依角色定位選擇：

- **攻擊型**：攻擊力、貫穿、對 BOSS 傷害、爆擊等
- **防禦型**：受擊觸發、迴避、減傷等
- **功能型**：攻速、移速、耐力、靈魂槽恢復等輔助效果

### 主題機率

- 主題別機率各20%
> 例：合成5個FOT勳章時，獲得FOT勳章之機率為100%。
> 例：合成3個FOT、1個SIN、1個SD勳章時，獲得勳章之機率為FOT 60%、SIN 20%、SD 20%。

### 類型機率

- 類型別機率各20%
> 例：合成3個攻擊型、1個防禦型、1個功能型勳章時，獲得勳章之機率為攻擊型60%、防禦型20%、功能型20%。

## PB 勳章（角色技能勳章）

PB 勳章是稀有度最高的 **Peta** 級勳章，依**各職業的技能**區分，**只能裝備、使用與自身相同職業的勳章**。它不吃型號套裝，而是直接強化指定技能。

### 插槽擴充

- 需先用「**PB 勳章插槽擴充券**」擴充時裝的**第 4 個插槽**，才能裝備 PB 勳章。
- 即使購買全新時裝，PB 勳章插槽也**不會預設為擴充狀態**，要自己開。

> [!warning]
> 已擴充 PB 勳章插槽的時裝**無法包裝**。

### 合成方法

PB 勳章只能透過「**PB 勳章合成器**」，以 **TB 勳章或 PB 勳章**作為材料合成取得。

- 必須用 TB／PB 等級的勳章**填滿 5 格**才能合成，也可以放超過所需格數的素材。
- 合成結果為**隨機**出現帶有各種職業專屬技能的 PB 勳章。

| 合成材料 | 佔用格數 |
| -------- | -------- |
| TB 勳章  | 1 格     |
| PB 勳章  | 2 格     |

> [!tip]
> 也可以用「**綁定型**」勳章合成，但綁定材料**只能和相同規格的道具一起合成**，且合成出的勳章**也會是綁定型**
> - 綁定 ＋ 綁定 ＝ 可合成
> - 綁定 ＋ 可交易 ＝ 不可合成

### 種類與效果

- **種類**：各角色的某個技能（隨機），項目很多這裡就不一一列出。
- **效果**：對應該枚 PB 勳章的**技能傷害 +10%**。

> [!important]
> 以下技能**不會**出現在 PB 勳章內（所有角色共通）
> - 全部 **EX1** 技能
> - 全部**二次覺醒**的新增技能
> - 部分**增益（buff）技能**
