本篇整理 Soulworker 的傷害計算流程，分成三大塊：**防禦減傷**、**玩家受擊**（Boss 打玩家）與 **玩家攻擊 Boss**。以下公式皆為社群實測／圖表轉錄。

## 符號說明

| 符號         | 意義                                |
| ------------ | ----------------------------------- |
| 防禦度       | 目標的防禦數值                      |
| 等級         | 攻擊者等級（通常取角色等級）        |
| 貫穿率       | 防禦貫穿百分比（0 ~ 1）             |
| 防禦減傷     | 由〈防禦減傷〉公式算出（0 ~ 1）     |
| 技能倍率     | 技能係數，例：300% → ×3             |
| B傷          | 增傷加成（傷害提升 %）              |
| 減少傷害     | 角色自身減傷 %                      |
| B減傷        | 特殊 Boss 傷害減免                  |
| BOSS減傷     | Boss 自身的減傷                     |
| 爆擊傷害     | 爆擊時額外附加的攻擊數值            |
| BOSS失敗率   | Boss 攻擊失敗（迴避）機率，預設 50% |
| 攻擊失敗減傷 | Boss 攻擊失敗時，玩家受到的額外減傷 |
| 攻擊失敗率   | 玩家攻擊未完全命中時的傷害倍率修正  |

## 防禦減傷

### 基礎公式

```
防禦減傷率 = 防禦度 ÷ (防禦度 + 等級 × 50)
```

### 含防禦貫穿

先用貫穿率折算「有效防禦」，再代回基礎公式：

```
有效防禦   = 防禦度 × (1 - 貫穿率)
防禦減傷率 = 有效防禦 ÷ (有效防禦 + 等級 × 50)
```

## 玩家受擊（Boss → 玩家）

### 一般受擊

```latex
受到傷害 = BOSS攻擊力 × (1 - 防禦減傷) × (1 - 減少傷害) × (1 - B減傷)
```

### 受擊迴避（Attack Fail）

Boss 攻擊失敗時，再乘上失敗率與對應減傷：

```
受到傷害 = BOSS攻擊力 × (1 - 防禦減傷) × (1 - 減少傷害) × (1 - B減傷)
         × BOSS失敗率 × (1 - 攻擊失敗減傷)
```

- 預設 `BOSS失敗率 = 50%`

## 玩家攻擊 Boss（玩家 → Boss）

### 命中（非爆擊）

```
造成傷害 = 攻擊力 × (1 - 防禦減傷) × (1 + B傷) × 技能倍率 × (1 - BOSS減傷)
```

### 命中且爆擊

爆擊時，攻擊力先加上爆擊傷害數值：

```
造成傷害 = (攻擊力 + 爆擊傷害) × (1 - 防禦減傷) × (1 + B傷) × 技能倍率 × (1 - BOSS減傷)
```

### 攻擊失敗（MISS）

```
造成傷害 = 攻擊力 × (1 - 防禦減傷) × (1 + B傷) × 技能倍率 × 攻擊失敗率 × (1 - BOSS減傷)
```

## 重點：乘區還是加區？

> [!warning] 「(1 + B傷)」與「技能倍率」是乘區，不是加區
> 很多人會直覺把兩者相加，但依本公式它們各自相乘。
>
> 例：B傷 +50% → ×1.5、技能倍率 300% → ×3
>
> - ✅ 正確：`攻擊 × 1.5 × 3`
> - ❌ 錯誤：`攻擊 × (1.5 + 3)`
>
> 否則傷害會膨脹得像某些遊戲策劃的 KPI。

> [!note] 待驗證
> 原圖術語有一處不一致：「MISS」實際上更接近「攻擊失敗／未完全命中倍率」，而非傳統意義上完全 0 傷的 MISS。建議日後以實機資料驗證。
