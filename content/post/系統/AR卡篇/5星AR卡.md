> [!tip]
> 5 星俗稱「神卡」，是畢業卡組的核心。但材料負擔極重，**不建議新手同時練多張**；先鎖定 1～2 張對自己流派最關鍵的（多半是對 BOSS 傷害或靈魂槽恢復）刻印起來。

## 被動

| 卡片 | 效果 | 備註 |
| ---- | ---- | ---- |
| 泰弗裡斯 | 提高對 BOSS 傷害／攻擊上升 | |
| 女王 | 命中時機率提高對 BOSS 傷害及爆傷，可疊四層 | |
| 凱薩琳 | 提高對 BOSS 傷害 | 不可與阿魯卡並用 |
| 阿魯卡 | 提高對 BOSS 傷害／貫穿 | 不可與凱薩琳並用 |
| 洛斯卡 | 間隔恢復靈魂槽及魔力 | 不可與艾歐並用 |
| 雷勒沛諾 | 間隔恢復耐力，並提高數秒攻擊力 | |
| 伯涅裡斯 | 提高攻速 | |
| 詩音 | 提高爆擊傷害 | |

| 卡片 | 效果 |
