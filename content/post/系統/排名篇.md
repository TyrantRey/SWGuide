## 排名系統是什麼

「排名」是依玩家在特定副本中的**通關表現（時間、次數、擊殺數等）** 換算分數，分數加總後產生名次，並依名次發放獎勵的賽季制度。

![](file-20260623143303746.png)
![](file-20260623143405882.png)
![](file-20260623143429711.png)
- **每個帳號僅有「代表角色」能參加排名**，代表角色可在角色選擇畫面設定。
- 賽季進行中若更換代表角色，**該賽季已累積的分數會直接消失**，請勿在賽季中途更換。

## 賽季與結算規則

| 項目   | 規則                             |
| ---- | ------------------------------ |
| 截止時間 | 每週一 00:00（GMT+8:00）排名截止        |
| 結算時間 | 截止後 2 小時發送排名獎勵，同時進入下一賽季        |
| 計算範圍 | 截止前 2 小時內的遊戲紀錄**不計入**本賽季排名     |
| 領獎時機 | 須等**下一個賽季開始後**，才能在排名視窗點擊「領取獎勵」 |
| 時裝獎勵 | 排名獎勵取得的時裝**僅可在當季使用**，賽季結束後自動刪除 |
| 賽季時長 | 一周                             |
- **單人副本排名**：分數以「同一角色」為單位結算，但帳號內所有角色的分數會合計成總排名。
- **組隊副本排名**：不分角色，統一結算（隊伍整體表現）。

## 計分與同分判定規則

- **統一所有排名副本的名次排序區間與對應配分**（不同副本不再各自一套配分標準）。

| 排名       | 分數 |
| ---------- | ---- |
| 1          | 100  |
| 2          | 99   |
| 3          | 98   |
| 4          | 97   |
| 5          | 96   |
| 6          | 95   |
| 7          | 94   |
| 8          | 93   |
| 9          | 92   |
| 10         | 91   |
| 11-20      | 89   |
| 21-30      | 87   |
| 31-40      | 85   |
| 41-50      | 80   |
| 51-前10%   | 75   |
| 前11%-20%  | 70   |
| 前21%-30%  | 65   |
| 前31%-40%  | 60   |
| 前41%-50%  | 55   |
| 前51%-70%  | 50   |
| 前71%-100% | 45   |

- 當結算分數相同時,下列排名分數較高的角色可列於前位
	1. 隱匿藏身處英雄
	2. 必然災禍
	3. 貝栗洛絲個人排名
- 詳細配分表與同分準則可在遊戲內「排名」選單 →「同分處理準則」查看。

### 已移除排名副本

#### 1. 深淵漫步

![](file-20260623144543429.png)

- 玩法：擊敗連續出現的怪物以累積分數；擊殺 Boss 級怪物後會連出下一隻 Boss，並延長計時 (類似公會副本)
- 入場後需啟動前方機械裝置，倒數開始後怪物才會出現
- 2024/11/21 改版已將「深淵漫步排名」與「組隊副本排名」整併移除

#### 2. 拉法酷巴

![](file-20260623144649132.png)

- 為 4 人組隊副本「隱匿藏身處」的單人版，部分機關佈置與怪物數值與組隊版不同。
- 2024/11/21 改為拉法庫巴 - 英雄

#### 3. 破滅救贖

![](file-20260623144811884.png)
- 2024/11/21 改為必然災禍

#### 4. 普雷瑪

![](file-20260623144912109.png)
- 2024/11/21 改為悲劇的妖精

#### 5. 虛空祭壇（組隊排名）

- 僅【專家】難度的關卡計入組隊排名，其他難度不計分。
- 2024/11/21 移除

#### 6. 墮落之月（組隊排名）

 - 2024/11/21 移除

#### 7. 天真之夢EP.2

- 2021/11/11 不再為排名副本
