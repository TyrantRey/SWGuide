靈魂小夥伴是什麼#

靈魂小夥伴就是這款遊戲的「寵物」系統最大的功能是可以自動拾取道具 ，跟著角色提供額外的數值與效果。它屬於後期內容，沒有捷徑，只能靠運氣與長期積累——急的話就只剩氪福袋或硬換碎片這條路。

筆記

靈魂小夥伴包含三塊：小夥伴本體（C／B／A／S 級）、工藝品（寵物裝備）、以及派遣。本體決定基礎，工藝品決定上限，派遣則是穩定的工藝品來源。

等級#

寵物分 C / B / A / S 四級，S 為最高。越高級的本體基礎屬性越好，但實戰價值仍要看具體寵物與你湊不湊得齊工藝品。

取得方式#

  • 抽取 / 福袋：主要來源，吃臉。
  • 碎片兌換：前日課孤獨的高原會掉工藝品，分解成無色寵物碎片；無色碎片可在碎片商店換指定寵物的碎片，換出一隻S 級需要約 90 個碎片（≈ 18 週）。
  • 派遣：寵物多起來後，派遣能額外產出工藝品，每週的兌換會越來越輕鬆。
